一块是Intel定位入门级的产品，一块是AMD主导高端市场的旗舰，两个似乎并没有多少交集的产品有对比的可能或者必要吗？可当我们尝试从全景生活体验出发，用单核论证处理器核心实用性时，两款产品真的出现了一些有趣的“化学反应”。

AMD全新的Ryzen系列处理器近日成为PC领域讨论的焦点，从跑分数据到功耗，都成为PC玩家讨论的焦点，率先登场的旗舰AMD Ryzen 7 1800X更是一口气提升到8核16线程的规格，令市场惊叹不已，可当人们留意PC实际使用处理器核心数量问题时，Ryzen系列处理器似乎在刻意引导“核心数量越多、跑分越高、处理器就更值得买”的逻辑。

实际上，AMD注重处理器多核心的原因在于其单核心性能表现并不足，难以同Intel同级产品对抗，设计与制造拆分之后，格罗方德的方案在单核性能上有短板，只能通过其他形式来弥补多核的整体性能。相对Intel 7代酷睿拥有“后发优势”的Ryzen系列处理器，很大可能本身就是针对不同市场定位的7代酷睿产品性能设计除的产品，通过核心数量和跑分数据压制，继续营造数字营销逻辑而已。

然而，面对长期在PC领域起主导地位的Intel而言，AMD整体实力还有欠缺。AMD极具针对性的推出Ryzen系列处理器产品，试图在每各市场区间取得优势时，产品线丰富的Intel完全可以随时调整自身产品线布局，推出类似Intel Core i3-7350K这样的明星产品来同AMD竞争。随着人们开始用实用性来考量购买需求时，处理器的单核性能开始成为关注焦点。

跑分数据对产品性能具有一定的参考价值，但产品实际性能最终还是需要回归到日常应用，毕竟绝大多数消费者不会是以跑分为数据购买PC产品的，因而测试的重点被放在日常图片/音频处理、PPT转PDF、游戏测试等模拟日常PC使用场景的测试上，选择GeForce GTX 1080这样的旗舰级显卡也是为了削弱显卡性能对整机测试成绩的影响，让测试结果尽量体现Intel Core i3-7350K和AMD Ryzen 7 1800X两款处理器的差异。

游戏是考验单核性能的最理想场景，正如文上所述，很少有专业玩家在游戏之时开启其他应用，总是希望在游戏中获得最佳实时性的体验。事实上，如果你并非游戏主播，需要在游戏同时进行视频转制、直播，那么很少会有多核应用。测试中我们选择了《魔兽世界》和《英雄联盟》两个极具代表性的网络游戏，在1080p的场景中，我们看到Intel Core i3-7350K领先了AMD Ryzen 7 1800X 5%~13%的成绩，而7350K的价格仅是后者的一半，这意味着在主流游戏的比拼中，Intel Core i3-7350K对PC整机性能的提升更为明显。

日常图片、音频转制、压缩处理是影音玩家常用到的功能。在TouchXPRT 2016的软件中，Intel Core i3-7350K和AMD Ryzen 7 1800X的三次平均值得分为343对307，Intel Core i3-7350K再次获得12%的领先优势，这代表用户使用Photoshop等软件处理图片时，Intel Core i3-7350K能让你节省10%的时间。

在所有的日常商务应用中，在Powerpoint中将PPT转存PDF较为消耗硬件资源的应用。我们特意选择了一个22MB的大文件作为转化对象，Intel Core i3-7350K只用14.4秒就完成了整个过程，而AMD Ryzen 7 1800X却用了16.9秒，相当于使用Intel Core i3-7350K进行PPT转PDF应用会节省14.8%的时间。

广大的苹果用户，均不能直接使用MP3格式3，所以需要把MP3转成AAC的格式，所以iTunes这个产品对很多苹果客户非常典型的应用， 10首MP3歌曲i3-7350K用了44.8秒，比对方的52.5秒快14.7%。此外，在《光影魔术手》、《QQ影音转换视频》等主流日常应用中，Intel Core i3-7350K依然拥有明显的优势。

还有一个值得玩味的数据，在采用风冷超频测试中，我们能将AMD Ryzen 7 1800X从3.6稳定超频到4.1；而Intel Core i3-7350K则可从4.2稳定超频到5.0（稳定超频的概念在于能够完成PCMark等测试软件的所以项目，且双拷中不死机）。前者超频比例约为13.8%；后者超频比例则达到约19%，这意味着尽管AMD号称所有型号不锁倍频，但其超频能力并非宣称那么强大。换一句话来说，英特尔留给用户的超频潜力更大，更具“性价比”。

移动终端市场，高通一则“多核无意义”的观点让不少砖家有被打脸的感觉，而PC领域，多核实用性也一直是人们讨论的焦点。当Windows 10成为主流操作系统，64位程序已经广泛应用时，多核、多线程在我们日常生活中究竟占了多大份量？

据我们统计，超过8成普通用户通常不会在PPT转PDF、音频格式转码或者压缩视频的同时，用PC玩游戏、看电影；除了需要直播的游戏主播外，绝大多数玩家在运行游戏时不会开启其他程序；而6成以上的普通用户在使用PC时倾向专一地浏览网页或文字输入，认为同步开启多项应用会影响每一项工作的质量……

事实上，人们绝大多数时候不会在一台PC上同步进行多项工作，虽然多核、多线程的处理器让PC能够胜任同步运行多个应用的需要，但用户注意力却很容易被过多的分支应用干扰，单线程的操控往往能更高效地完成工作或者作品，这使得PC绝大多数时间实际上只会调用一个处理器核心、进行一个进程。正因如此，今天我们所做的测试就变得极有意义。

无论是1499元的售价还是带“K”的命名，Intel Core i3-7350K都应该是一款面向中端主流市场的产品，可就是这样一款主流走量的产品，却在单核性能上、特别是主流游戏之中近乎全面压制了旗舰定位的AMD Ryzen 7 1800X，再一次证明AMD处理器的确在单核心性能上难以同Intel一争高低。对于众多主流用户、普通游戏玩家而言，当双核四线程的Intel Core i3-7350K完全能够满足日常应用所需时，为何不将多出来的预算放到显卡、硬盘、内存或其它配件上呢？这对于AMD而言恐怕并不是一个好消息，定位顶端的AMD Ryzen 7 1800X尚不能在单核应用上获得优势，面对主流市场、尚未亮相的Ryzen 5、Ryzen 3将如何在单核上与英特尔抗衡？让我们拭目以待！

无论是购买处理器还是选择PC配置，性价比是出现频率非常高的词语，但有多少人思考过如何定义自己的性价比呢？如果你并非“GDP”（“G”即Gamers，指高端游戏玩家；“D”即Design Workers，指设计行业从业者；“P”即Photographers，指摄影爱好者、职业摄影师等）这样的高性能需求人群，那么请慎重考虑你为核心付出的多余成本。
