对于 Windows 和 Mac OS X 到底谁更好的话题从来都没有停过，而再深入一些的就会提到为什么买苹果电脑要装Windows系统，其实这里面有很多大家对苹果电脑的误解和错误的认识。

Windows 系统相信大家再熟悉不过了，所以我们先来说说 Mac OS X，这个只能安装在苹果电脑上的PC系统其实很多人并不熟悉，大家对它的鄙视其实大多都来自于用苹果电脑的人，而并非这款系统本身。

作为一位 iMac 的老用户，我简单说一下我为什么会选择苹果电脑，当然我还同时在用一款搭载 Windows 系统的宏碁笔记本电脑。

Mac OS X 的优势非常明显，它操作简单，几乎不用去操心任何设置或选项，买回来可以直接上手操作，甚至什么杀毒软件从来都没有装过。

这与 Windows 系统有着完全不同的套路，往往买回一台新的 Windows 电脑第一件事就是装杀毒软件，然后要各种支持软件，非常麻烦，甚至需要折腾一整天，而苹果电脑几乎可以直接使用。

还有一个优势就是极小的故障率。用 Windows 电脑经常会出现一些莫名其妙的小问题，有的时候硬件还会出问题，这使得人们还要自己想办法维修，我自用过3台 Windows 笔记本电脑，完全没有苹果电脑那样的体验。

并不能这么说，因为两款系统都有自己擅长的领域。传统的银行、办事处等地方仍然是 Windows系统，甚至有些还在用最老的 Windows 95 系统。公司的票据打印系统有的只支持 Windows系统，所以想用苹果电脑也不行。

Windows 系统仍然是办公领域支持最好的系统。而 Mac OS X 则是对某些专业领域更加擅长，如音频、视频、编程等等。

以瓦力小伙伴做视频为例，在 Windows 电脑上剪辑一段评测视频可能要10个小时，而在苹果电脑上可能只需要3个小时，主要是很多 Mac OS X 上的软件非常方便，很多特效全都是现成的。

Mac OS X 和我们手机上的 iOS 类似，它的应用软件全都是经过严格审核才上线的，所以不管是应用质量，还是功能全都比较完善，并且适配完美。

另外我们几乎很少在苹果电脑上看到莫名多出来的垃圾软件，但这在 Windows 电脑上却司空见惯了，一天不出现都新鲜，这也是 Windows 不如 Mac OS X 地方。

其实 Windows 和 Mac OS X 并没有绝对的好与坏，只是一个使用习惯的问题，而辩论到底谁好谁坏的话题也应该停止了。

买了苹果电脑装 Windows 系统的无非两种人，一是确实工作需要使用 Windows 电脑，但自己又喜欢苹果电脑的样子。二是买来装X，自己不会用 Mac OS X ，且连去学一下的耐心都没有。

但苹果电脑装 Windows 系统，其实对电脑本身有非常不好的作用，如：系统适配的流畅度，系统对每个硬件的智能管理，续航时间和散热控制全都会有影响，从爱惜电脑的角度上来讲，还是尽量不要这么玩。

另外还有一种套路就是“黑苹果”，这种电脑可以在普通电脑上模拟出一个 Mac OS X 系统，让你可以在别的电脑上使用苹果电脑上的那些专业软件，不过同样的这并不是最完美的适配效果。

从使用习惯上来看， Mac OS X 的文件管理，甚至窗口功能按钮的位置全都是不一样的，鼠标的操作也是完全不一样的。所以第一次使用苹果电脑的人基本上没法正常“娱乐”，还要适应一下才可以。

并不是，其实 Mac OS X 和 Windows 一样，大多应用都是有破解版的，如果你想找免费的，你可以这么搜索：Final Cut Pro for mac，这个“for mac”很重要哦。

并不是，如果你是在系统内的应用商店下载，点一下就可以直接安装了，和手机一样容易。如果你是自己下载的，和 Windows 一样，双击安装即可，下一步/下一步/下一步......

并不是，Mac OS X 也有病毒，只不过相对来说少一点，毕竟应用商店审核还是很严格的，中病毒多为第三方下载平台的应用。

只有一半对，其实那些原生支持Mac OS X 的游戏还是很牛逼的，比如之前一直很火的《炉石传说》，我用iMac 玩的时候，全都是超清分辨率，音效简直是爽爆了，在手机和 Windows 上就没有这样的体验。

当然，移植的游戏确实很垃圾，毕竟苹果电脑其实并不擅长娱乐，更多的还是在专业领域。

这也只能说对了一半，苹果电脑的触控板手势操作确实非常牛，很多操作都可以不用鼠标来完成。我曾在发布会现场见到一个高手剪辑视频，全都是触控板操作，太帅了。

相对的 Windows 对触控板的支持就少了很多，而用 Windows 笔记本的人大多也只是发发邮件或简单的办公罢了。

总结：其实不管是 Windows 还是 Mac OS X ，都是用来服务我们的，只不过他们偏向的领域和重心不同，这也是两款系统的发展方向，我们没有必要去纠结到底买哪一款产品，只要你喜欢就好。

这就像买车，如果你经常上山肯定要买 SUV，如果跑公路买什么车都行。就算你喜欢法拉利，但你上山的话也是力不从心。
